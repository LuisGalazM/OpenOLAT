The package properties is used for collecting different properties in one place insted of generating for each properties a new table entry in the database.

Use it as a generic collection of properties for security reasons on users, groups, courses...

Below is a list of actions and their corresponding descriptions, detailing various workflow and task management activities in assessment, which are extracted from the LOG files:

Action Possible Detail Description Datatype
ASSESSMENT_ID set to: <id> Anomaly action, sets assessment identifier. int
ATTEMPTS set to: <int> Records how many attempts have been made. int
Allow reset task of <filename> Allows the user to reset the specified task. string
Assignment of <filename> Assigns a specific task or document. string
Back to submission of <filename> Reverts the task back to submission status. string
COMMENT set to: <comment> A comment was set. string
Collect of <filename> Collects a specific document. string
Collect revision of <filename> Collects a revision of a specific document. string
Corrections of <filename> Submits corrections or updates for a document. string
Deadline extension for <filename> Extends the deadline for a specific assignment. string
Evaluation deleted - Deletes an evaluation. -
Evaluation finshed - Marks evaluation as finished (typo error in original logs). -
Evaluation reopened - Reopens an evaluation for further action. -
Evaluation started - Marks the start of an evaluation. -
FILE ADDED: <filename> A file was added (dropbox). string
FILE DELETED: <filename> A file was deleted. string
FILE UPLOADED: <filename> A file was uploaded. string
Peer reopen of <filename> The file got reopened for peer review. string
Peer review of <filename> Invalidates or performs peer review on the file. string
Reopen peer-review of <filename> Reopens a previously submitted peer-review task or document. string
Reset task of <filename> Resets progress of a specific task. string
Review of <filename> A review was performed on the document. string
Revision of <filename> Handles revisions for a specific document. string
Submit of <filename> Submits the specified document or task. Could be either upload, update, delete, create or edit - see SubmitEvent class. string
assessment document added: <filename> A document was added for assessment purposes. string
assessmentId set to: <id> Sets the assessment identifier. int
grade set to: <grade> A grade value was assigned. double/string
passed set to: <status> Sets the status to true, false, or undefined. boolean/string
reset course element - Resets the assessment or course element. string
score set to: <score> Assigns or updates the score. double/null
